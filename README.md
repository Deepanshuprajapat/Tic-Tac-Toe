# ❌⭕ Tic Tac Toe Game

A simple **Tic Tac Toe** game built using Python and Tkinter GUI.

Players take turns placing **X** and **O** on a 3x3 grid.  
The first player to align 3 symbols in a row, column, or diagonal wins the game.

---

## 🎮 Features

- GUI-based Tic Tac Toe game
- Two-player gameplay
- Winner detection
- Tie detection
- Winning row highlight
- Simple and beginner-friendly code

---

## 🛠 Requirements

- Python 3.x
- Tkinter (comes pre-installed with Python)
